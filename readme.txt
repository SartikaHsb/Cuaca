Langkah yang harus dilakukan adalah:
. Install laravel, composer (follow this link : https://laravel.com/docs/5.3/installation)
. Ekstrak folder yang didownload
. Buka cmd dan arahkan path cmd ke folder yang telah diekstrak tersebut
. Ketik php artisan serve. Perintah untuk menjalankan website
. Buka browser dan ketikkan localhost:8000
